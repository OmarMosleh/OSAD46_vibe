# Sorting Algorithm Visualizer

An interactive web application that visualizes six different sorting algorithms with colorful animations. Built with pure vanilla JavaScript, HTML, and CSS - no frameworks or libraries!

## 🎯 Features

- **6 Sorting Algorithms**: Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, Quick Sort, and Heap Sort
- **Interactive Controls**: Adjust speed, array size, and choose between random or custom input
- **Real-time Statistics**: Track comparisons and swaps as they happen
- **Color-Coded Visualization**:
  - 🟢 Green: Sorted elements
  - 🟠 Orange: Comparing elements
  - 🔴 Red: Swapping elements
  - 🟦 Teal: Default/unsorted elements
- **Smooth Animations**: Watch algorithms sort step-by-step with adjustable speed
- **Responsive Design**: Works on different screen sizes

## 🚀 Getting Started

Simply open `index.html` in your web browser - no installation or build process required!

```bash
# Clone the repository
git clone https://github.com/OmarMosleh/OSAD46_vibe.git

# Open in browser
cd OSAD46_vibe
open index.html
```

## 🎮 How to Use

1. **Select an Algorithm**: Choose from 6 different sorting algorithms in the dropdown
2. **Set Array Size**: Choose how many elements to sort (5-100)
3. **Adjust Speed**: Use the slider to control animation speed (1-100)
4. **Generate Data**:
   - Click "Generate Random" for a random array
   - Click "Custom Input" to enter your own numbers
5. **Start Sorting**: Click "Start Sorting" to watch the algorithm work
6. **Stop Anytime**: Click "Stop" to halt the sorting process

## 📚 Algorithms Included

- **Bubble Sort** - O(n²) - Simple comparison-based algorithm
- **Selection Sort** - O(n²) - Finds minimum element repeatedly
- **Insertion Sort** - O(n²) - Builds sorted array one element at a time
- **Merge Sort** - O(n log n) - Divide and conquer approach
- **Quick Sort** - O(n log n) - Fast partitioning algorithm
- **Heap Sort** - O(n log n) - Binary heap-based sorting
